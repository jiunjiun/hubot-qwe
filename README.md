# hubot-qwe

A hubot script that does the things

See [`src/qwe.coffee`](src/qwe.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-qwe --save`

Then add **hubot-qwe** to your `external-scripts.json`:

```json
["hubot-qwe"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
